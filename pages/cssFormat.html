Colores

Elemento

Descripcion

Documentacion
Usando nombres de colores Podemos especificar un color usando un nombre predefinido. Documentacion
Usando la funcion RGB (r,g,b) Para definir un color como la mezcla las intensidades de los colores primarios podemos usar la función: rgb(red, green, blue), Los valores posibles son: [0-255] o [0%-100%] Documentacion
Usando valores hexadecimales Podemos utilizar un número hexadecimal para definir un color. Es una alternativa a la función rgb, es decir permite definir un color como la mezcla de intensidades de los colores primarios. Documentacion
Opacidad La propiedad CSS opacity define la transparencia de un elemento, esto es, en qué grado se superpone el fondo al elemento. Documentacion
Usando la función hsl(h, s, l) Podemos definir el color usando la función hsl(hue, saturation, lightness). Documentacion
hue Es el color Los valores posibles son: [0-360] Podemos obtener los valores de la rueda de colores. Documentacion
saturation Determina que tan fuerte o vivo es un color Los valores posibles son: [0%-100%] Donde 100% significa completamente saturado (color muy vivo) Donde 0% significa sin saturación alguna (color apagado) Documentacion
lightness Permite establecer la claridad u oscuridad de un color Los valores posibles son de: [0%-100%] Donde 100% significa completamente blanco (brillantez total) Donde 50% es normal Donde 0% es competamente negro (sin brillantez) Documentacion
Usando la función hsla(h, s, l, a) Podemos definir el color usando la función hsla(hue, saturation, lightness, opacity), con un valor de opacidad. Documentacion

Selectores

Elemento

Descripcion

Documentacion
SELECTOR UNIVERSAL (*) Selecciona todos los elementos: *{ declaraciones CSS } Documentacion
SELECTORES DE TIPO (elem) Permite seleccionar elementos usando el nombre del elemento: elem{ declaraciones CSS } Documentacion
SELECTOR DE CLASE (.elem) Permite seleccionar elementos en función de los valores del atributo class de los elementos. Para que los elementos sean seleccionados, su atributo class debe incluir un valor que coincida con el valor del selector. .elem { declaraciones CSS } Documentacion
SELECTORES DE ID (#elem) Permite seleccionar un elemento en función del valor del atributo ID del elemento. Para que el elemento sea seleccionado, su atributo id debe coincidir exactamente con el valor dado en el selector.
#valor_atributo { declaraciones CSS } 		Documentacion
SELECTORES DE ATRIBUTO
(selector[attr]) Permite seleccionar elementos en función de la presencia o del valor de un determinado atributo. --> Selecciona elementos que tengan un atributo con el nombre attr Documentacion
(selector[attr=value]) Selecciona elementos que tengan un atributo con el nombre attr. El único valor del atributo es exactamente value.
(selector[attr~=value]) Selecciona los elementos que tengan un atributo con el nombre attr. Uno de los valores del atributo es exactamente value.
(selector[attr ^= value]) Selecciona los elementos que tengan un atributo con el nombre attr. El primero de los valores del atributo empieza con value.
(selector[attr $= value]) Selecciona los elementos que tengan un atributo con el nombre attr. El último de los valores del atributo termina con value.
(selector[attr *= value]) Selecciona los elementos que tengan un atributo con el nombre attr. Uno de los valores del atributo contiene al menos una ocurrencia de value.
Pseudo Clase Una pseudoclase CSS es una palabra clave que se añade a los selectores y que especifica un estado especial del elemento seleccionado. permiten aplicar un estilo a un elemento no sólo en relación con el contenido del árbol de documento, sino también en relación a factores externos Documentacion
:disabled Selecciona elementos que tengan un atributo con el nombre attr. El único valor del atributo es exactamente value. Documentacion
:first-child La pseudo-clase :first-child de CSS representa el primer elemento entre un grupo de elementos hermanos. Documentacion
:nth-child La pseudo-clase :nth-child() de CSS coincide con uno o más elementos en función de su posición entre un grupo de hermanos. Documentacion
:hover La pseudo-clase :hover de CSS coincide cuando el usuario interactúa con un elemento con un dispositivo señalador, pero no necesariamente lo activa. Generalmente se activa cuando el usuario se desplaza sobre un elemento con el cursor Documentacion
PSEUDO- ELEMENTOS Seleccionan parte de un elemento en lugar del elemento mismo. Los pseudo-elementos se añaden a los selectores y permiten añadir estilos a una parte concreta de los elementos seleccionados. Documentacion
::after crea un pseudo-elemento que es el último hijo del elemento seleccionado. Es comunmente usado para añadir contenido cosmético a un elemento con la propiedad content.Es en linea (inline) de forma predeterminada. Documentacion
::before crea un pseudoelemento que es el primer hijo del elemento seleccionado. Es usado normalmente para añadir contenido estético a un elemento, usando la propiedad content. Este elemento se muestra en línea con el texto de forma predeterminada. Documentacion
::first-letter plica estilos a la primera letra de la primera línea un elemento de bloque, sólo cuando no es precedido de otro contenido (como imágenes o tablas). Documentacion
::first-line aplica estilos a la primera línea de un elemento de bloque. Nótese que la longitud de la primera línea depende de muchos factores, incluyendo el ancho del elemento, el ancho del documento y el tamaño de fuente del texto. Documentacion
::selection aplica reglas a una porción de un documento que ha sido destacado (por ejemplo: selección con el mouse o algún otro puntero en un dispositivo) del usuario. Documentacion
::backdrop Este pseudo-elemento es una caja que se muestra inmediatamente debajo del elemento (y sobre el elemento inmediatamente inferior de la pila, si es que hay), dentro de dicha capa superior. Documentacion
Union de selectores
LISTA DE SELECTORES Selecciona la unión de los elementos seleccionados por cada selector de la lista: Selector-1, Selector-2, … { } Documentacion
Combinaciones de selectores
ELEMENTOS DESCENDIENTES A B {} Donde A y B son selectores Seleccionamos todos los elementos seleccionados por B que son descendientes de los elementos seleccionados por A Documentacion
ELEMENTOS HIJOS A>B{} Donde A y B son selectores Seleccionamos todos los elementos seleccionados por B que son hijos de los elementos seleccionados por A
ELEMENTOS HERMANOS A~B {} Donde A y B son selectores. Seleccionamos todos los elementos seleccionados por B que son hermanos de los elementos seleccionados por A
ELEMENTOS HERMANOS ADYACENTES A+B {} Donde A y B son selectores. Seleccionamos todos los elementos seleccionados por B que son hermanos y que están inmediatamente despues de los elementos seleccionados por A. Selecciona el elemento B solo si sigue inmediatamente a A. Los elementos A y B son hermanos.
ESPECIFICIDAD La especificidad es la manera mediante la cual los navegadores deciden qué valores de una propiedad CSS son más relevantes para un elemento y, por lo tanto, serán aplicados. La especificidad está basada en las reglas de coincidencia que están compuestas por diferentes tipos de selectores CSS. Documentacion
Herencia Algunos valores de las propiedades CSS que se han establecido para los elementos padre los heredan los elementos hijo, pero otros no. Documentacion

Unidades

Elemento

Descripcion

Documentacion
Integer Un integer es un número entero, como 1024 o -55. Documentacion
Number Un number representa un número decimal; puede tener o no un punto de separación decimal con un componente fraccionario, por ejemplo: 0.255, 128 o -1.2.
Dimension Una dimension es un number con una unidad asociada, por ejemplo: 45deg (grados), 5s (segundos) o 10px (píxeles). dimension es una categoría general que incluye los tipos length, angle, time y resolution.
Percentage Un percentage representa una fracción de algún otro valor, por ejemplo, 50%. Los valores de porcentaje siempre son relativos a otra cantidad, por ejemplo, la longitud de un elemento es relativa a la longitud de su elemento padre.
Vw Este valor es relativo al ancho de la ventana gráfica, por lo que 10vw es el 10 por ciento del ancho de la ventana gráfica. Si cambiases el ancho de la ventana de tu navegador, el tamaño de la caja cambiaría.
Vh 1% de la altura de la ventana gráfica.
